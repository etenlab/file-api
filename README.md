## Description

File Api - This api upload files to aws-s3 bucket and store the links to our database.

## Federation

Not sure, but file-api doesn't work properly, so cancelled federation mode.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
